# 🔐 Sistema de Login

Aplicação desktop desenvolvida em Python que simula um sistema de login com validação de credenciais.

## 🖥️ Como funciona
- Digite seu usuário e senha
- Clique no botão Login
- O sistema exibe feedback visual — verde para sucesso, vermelho para erro

## 🛠️ Tecnologias utilizadas
- Python
- CustomTkinter (interface gráfica moderna)

## 📚 Sobre
Projeto desenvolvido durante minha transição de carreira de UX Design para Engenharia de Software.
